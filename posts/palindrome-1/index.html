Algorithm - Palindrome(I)

Given a string, if it reads the same forward and backward. We think this string is a palindrome.

So how to check a string whether it is a palindrome or not?

Solution:

func isPalindrome(_ s: String) -> Bool {
    // 1
    let array = Array(s)

    // 2
    var left = 0
    var right = array.count - 1

    // 3
    while left < right {
        // 4
        if array[left] == array[right] {
            left += 1
            right -= 1
        } else {
            return false
        }
    }

    // 5
    return true
}

How it works:

  1. Create a characters array
  2. Define two variables: left or right. We use left variable to track the location from left, and use right variable to track location from right.
  3. We check whether left location meet the right location.
  4. Everytime we compare the element at the left location is the same with the element at the right location. If equals, then left location move one step to right, and right location move one step to left. If not, return fasle(means this string is not a palindrome).
  5. If the left location meet the right location, and all the elements are the same, return true(means this string is a palindrome).
